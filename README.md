# Dharshan R - Portfolio

A modern, responsive portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- **Dark/Light Mode** - Theme toggle with system preference support
- **Smooth Scroll** - Native smooth scrolling for navigation
- **Animations** - Subtle fade-in animations using Framer Motion
- **Project Cards** - Hover lift effect on project cards
- **Typing Animation** - "Building scalable digital experiences." in the Hero
- **Mobile-First** - Fully responsive design

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3
- Framer Motion 11

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:5175` to view the portfolio.

## Build for Production

```bash
npm run build
npm run preview
```
